imagecluster.io.timestamp¶

imagecluster.io.timestamp(filename, source='auto')[source]¶

Read single timestamp for image in filename.

Parameters:

filename : str

source : {‘auto’, ‘stat’, ‘exif’}

Read timestamps from file stats (‘stat’), or EXIF tags (‘exif’). If ‘auto’, then try ‘exif’ first.
Returns:

float

timestamp, seconds since Epoch
