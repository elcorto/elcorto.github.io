pwtools.rbf.RBFInt.get_dist_mat¶

RBFInt.get_dist_mat(X=None, C=None)[source]¶

Matrix of distance values r_ij = ||x_i - c_j|| with:

x_i : X[i,:]
c_i : C[i,:]
Parameters:

X : None or array (M,N) with N-dim points

Training data or interpolation points. If None then self.Rsq is returned.

C : 2d array (K,N), optional

If None then self.C is used.
Returns:

Rsq : (M,K), where K = M usually for training
