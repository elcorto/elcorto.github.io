PDBFile.is_set_attr(attr)¶

Check if self has the attribute self.<attr> and if it is _not_ None.

Parameters:

attr : str

Attrubiute name, e.g. ‘foo’ for self.foo
Returns:

True : attr is defined and not None

False : not defined or None
