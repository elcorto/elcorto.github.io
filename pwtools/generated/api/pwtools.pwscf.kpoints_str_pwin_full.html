pwtools.pwscf.kpoints_str_pwin_full¶

pwtools.pwscf.kpoints_str_pwin_full(lst, shift=[0, 0, 0], gamma=True)[source]¶

Full k-points string for pw.x input files, card K_POINTS.

Parameters:

lst : sequence (3,)

shift : sequence (3,), optional

gamma : bool, optional

If lst == [1,1,1] then return “K_POINTS gamma”, else “K_POINTS automatic <newline> 1 1 1 <shift>”.
