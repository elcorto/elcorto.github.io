pwtools.timer.TagTimer.pt¶

TagTimer.pt(tag, msg='')[source]¶

Print time difference since last t(tag) call for tag, which is self.time_ar_dict[tag][1] - self.time_ar_dict[tag][0].

Parameters:

tag : anything hashable

a tag (most likely a string) associated with a storage array

msg : string

Extra string to be printed along with the time difference.
