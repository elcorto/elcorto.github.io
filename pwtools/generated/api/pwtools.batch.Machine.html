pwtools.batch.Machine¶

class pwtools.batch.Machine(hostname=None, subcmd=None, scratch=None, home=None, template=None, filename=None)[source]¶

Bases: object

This is a container for machine-specific stuff. Most of the machine-specific settings can (and should) be placed in the corresponding job template file. But some settings need to be in sync between files like the scratch dir, the $HOME etc.

Useful to predefine commonly used machines.
