pwtools.test.tools.DictWithAllTypesFactory¶

class pwtools.test.tools.DictWithAllTypesFactory(comp_func=None)[source]¶

Bases: object

Factory for creating functions which can compare dicts with values of “any” type, also numpy arrays. Nested dicts are possible.
