pwtools.calculators.Lammps¶

class pwtools.calculators.Lammps(**kwds)[source]¶

Bases: ase.calculators.calculator.FileIOCalculator, pwtools.calculators.CalculatorBase

LAMMPS calculator.

Examples

Define a calculator object:

>>> calc = Lammps(label='/path/to/calculation/dir/lmp',
...               pair_style='tersoff',
...               pair_coeff='* * /path/to/potential/dir/AlN.tersoff Al N',
...               command='lammps < lmp.in > lmp.out 2>&1',
...               ) 
>>> at=crys.Structure(...).get_ase_atoms(pbc=True)
>>> at.set_calculator(calc)
>>> at.get_potential_energy()
