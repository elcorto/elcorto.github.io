pwtools.comb.unique2d¶

pwtools.comb.unique2d(arr, what='row')[source]¶

Reduce 2d array arr to a 2d array with unique rows (or cols).

Parameters:

arr : 2d-like

what : str

{‘row’, ‘col’}
Returns:

numpy 2d array

Notes

# These do the same: >>> unique2d(permute(a)) # uses more memory, slower >>> permute(a, skip_equal=True))

Examples

>>> a=array([[1,2,3], [1,2,3], [1,2,4]])
>>> unique2d(a, 'row')
array([[1, 2, 3],
       [1, 2, 4]])
