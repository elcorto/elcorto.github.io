pwtools.rbf.RBFInt.__call__¶

RBFInt.__call__(*args, **kwargs)[source]¶

Call self.interpolate() or self.deriv().

Parameters:

X : 2d array (L,N) or 1d array (N,) or None

L data points in N-dim space. If None, then X = self.X, i.e. just evaluate the training set. If 1d, then this is just 1 point, which will be converted to shape (1,N) internally.

der : int

If == 1 return matrix of partial derivatives (see self.deriv()), else interpolated Y values (default).
Returns:

YI : 1d array (L,)

Interpolated values.

or

D : 2d array (L,N)

1st partial derivatives.
