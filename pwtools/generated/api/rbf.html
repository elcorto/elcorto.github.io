rbf¶

Radial Basis Function N-dim fitting. See Radial Basis Functions Networks for interpolation or fitting of N-dim data points for details.

FitError(points, values[, rbf_kwds, cv_kwds]) Direct or cross-validation (CV) fit error of Rbf for a parameter set [p,r] or just [p].
Rbf(points, values[, centers, rbf, r, p, …]) Radial basis function network interpolation and fitting.
estimate_p(points[, method]) Estimate \(p\).
fit_opt(points, values[, method, what, …]) Optimize \(p\) or \((p,r)\) using a cross validation error metric or the direct fit error if cv_kwds is None.
rbf_dct
rbf_gauss(rsq, p) Gaussian RBF \(\exp\left(-\frac{r^2}{2\,p^2}\right)\)
rbf_inv_multi(rsq, p) Inverse Multiquadric RBF \(\frac{1}{\sqrt{r^2 + p^2}}\)
rbf_multi(rsq, p) Multiquadric RBF \(\sqrt{r^2 + p^2}\)
