pwtools.sql.SQLiteDB.get_header¶

SQLiteDB.get_header(table=None)[source]¶

Return the header of the table:

Parameters:table : str, optional

Examples

>>> db = SQLiteDB('test.db')
>>> db.execute("create table foo (a text, b real)")
>>> db.get_header('foo') 
[('a', 'text'), ('b', 'real')]
