pwtools.num.meshgridt¶

pwtools.num.meshgridt(x, y)[source]¶
A version of
X,Y = numpy.meshgrid(x,y)

which returns X and Y transposed, i.e. (nx, ny) instead (ny, nx) where nx,ny = len(x),len(y).

This is useful for dealing with 2D splines in scipy.interpolate.bisplev(), which also returns a (nx,ny) array.

Parameters:x,y : 1d arrays
