pwtools.common.PydosConfigParser¶

class pwtools.common.PydosConfigParser(defaults=None, dict_type=<class 'collections.OrderedDict'>, allow_no_value=False)[source]¶

Bases: ConfigParser.SafeConfigParser

All values passed as arg to self.set(self, section, option, arg) are converted to a string with frepr(). get*() methods are the usual ones provided by the base class ConfigParser.SafeConfigParser: get(), getint(), getfloat(), getboolean(). Option keys are case-sensitive.
