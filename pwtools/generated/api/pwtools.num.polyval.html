pwtools.num.polyval¶

pwtools.num.polyval(fit, points, der=0, avg=False)[source]¶

Evaluate polynomial generated by polyfit() on points.

Parameters:

fit, points : see polyfit()

der : int, optional

Derivative order. Only for 1D, uses np.polyder().

avg : bool, optional

Internal hack, only used by avgpolyval().

See also

PolyFit, PolyFit1D, polyfit()

Notes

For 1D we provide “analytic” derivatives using np.polyder(). For ND, we didn’t implement an equivalent machinery. For 2D, you might get away with fitting a bispline (see Interpol2D) and use it’s derivs. For ND, try rbf.py’s RBF interpolator which has at least 1st derivatives for arbitrary dimensions.
