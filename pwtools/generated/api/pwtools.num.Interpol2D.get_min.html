pwtools.num.Interpol2D.get_min¶

Interpol2D.get_min(x0=None, **kwds)[source]¶

Return [x,y] where z(x,y) = min(z) by minimizing z(x,y) w/ scipy.optimize.fmin().

Parameters:

x0 : sequence, length (2,), optional

Initial guess. If None then use the data grid point with the smallest z value.
Returns:

[xmin, ymin]: 1d array (2,)
