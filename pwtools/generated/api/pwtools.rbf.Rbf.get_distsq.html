pwtools.rbf.Rbf.get_distsq¶

Rbf.get_distsq(points=None)[source]¶

Matrix of distance values \(R_{ij} = |\mathbf x_i - \mathbf c_j|\).

\(\mathbf x_i\) : points[i,:] (points)
\(\mathbf c_j\) : self.points[j,:] (centers)
Parameters
pointsarray (K,N) with N-dim points, optional

If None then self.points is used (training points).

Returns
distsq(M,K), where K = M for training
