pwtools.comb._swap¶

pwtools.comb._swap(arr, i, j)[source]¶

Swap arr[i] <-> arr[j]. Return i-j-swapped copy of arr.

Parameters:arr : 1d list

Notes

Must return a copy of arr b/c the swap is an in-place operation and lists are mutable.
