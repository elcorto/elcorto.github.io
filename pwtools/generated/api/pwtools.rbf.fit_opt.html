pwtools.rbf.fit_opt¶

pwtools.rbf.fit_opt(points, values, method='de', what='pr', cv_kwds={'nr': 1, 'ns': 5}, opt_kwds={}, rbf_kwds={})[source]¶

Optimize \(p\) or \((p,r)\) using a cross validation error metric or the direct fit error if cv_kwds is None.

Parameters

points, values : see Rbf

method : str

‘de’ : scipy.optimize.differential_evolution()
‘fmin’: scipy.optimize.fmin()

what : str

‘p’ or ‘pr’

cv_kwds, rbf_kwds : see FitError

opt_kwds : dict

kwds for the optimizer (see method)
