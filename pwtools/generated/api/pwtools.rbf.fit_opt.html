pwtools.rbf.fit_opt¶

pwtools.rbf.fit_opt(points, values, method='de', what='pr', cv_kwds={'n_repeats': 1, 'n_splits': 5}, opt_kwds={}, rbf_kwds={})[source]¶

Optimize Rbf’s hyper-parameter \(p\) or both \((p,r)\).

Use a cross validation error metric or the direct fit error if cv_kwds is None. Uses FitError.

Parameters
points, valuessee Rbf
methodstr
‘de’ : scipy.optimize.differential_evolution()
‘fmin’: scipy.optimize.fmin()
‘brute’: scipy.optimize.brute()
whatstr

‘p’ (optimize only p and set r=None) or ‘pr’ (optimize p and r)

cv_kwds, rbf_kwdssee FitError
opt_kwdsdict

kwds for the optimizer (see method)

Returns
rbfiRbf

Rbf instance initialized with points, values and found optimal p (and r).
