pwtools.kpath.get_path_norm¶

pwtools.kpath.get_path_norm(ks)[source]¶

Like in QE’s plotband.f90, path_norm = kx there. Return a sequence of cumulative norms of the difference vectors which connect each two adjacent k-points.

Parameters

ks (array (nks, 3)) – array with nks k-points on the path
