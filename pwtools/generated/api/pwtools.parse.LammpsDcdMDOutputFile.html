pwtools.parse.LammpsDcdMDOutputFile¶

class pwtools.parse.LammpsDcdMDOutputFile(*args, **kwds)[source]¶

Bases: pwtools.parse.DcdOutputFile, pwtools.parse.LammpsTextMDOutputFile

Parse Lammps DCD binary output + log.lammps text output.

Hardcodes files:
dcdfilename = basedir/lmp.out.dcd

Notes

  • cell: The DCD file format stores only cryst_const (see unitcell in dcd.f90). Trajectory.cell calculated from Trajectory.cryst_const is aligned in the same way as LammpsTextMDOutputFile.cell. That’s why the cell and cryst_const obtained from read_lammps_md_dcd() and read_lammps_md_txt() must be identical up to numerical noise (about 1e-6 for default lammps text printing precision).
